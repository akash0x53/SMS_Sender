SMS Sender Utility

NOTE: Works on Nokia, Samsun or any other which supports AT commands
             (Tested on Nokia, Samsung [exclude android])	
OS: tested on Windows XP (Not for linux :-( )

1) You need to download some files from web
	a. Download the javax.comm package as a ZIP file. 
	b. Unzip it into C:\j2sdkXXX, resulting in C:\j2sdkXXX\commapi

	c. Copy win32comm.dll to C:\j2sdkxxx\jre\bin
	d. Copy comm.jar to C:\j2sdkxxx\jre\lib\ext
	e. Copy javax.comm.properties to C:\j2sdkxxx\jre\lib

	f. Copy win32comm.dll to C:\j2sdkXXX\bin
	g. Copy comm.jar to C:\j2sdkXXX\lib\ext
	h. Copy javax.comm.properties to C:\j2sdkXXX\lib

2) Install mobile phone drivers 
3) connect mobile to USB port 
4) goto Control panel ->phone & modem, find your device there; note down port number and enter same number in port.txt file
5) open center.txt file. Enter SMS center number; you can find it in your SMS settings
6)run code... :) 
done !!! :)
